Lesson 1: Variables

JavaScript variables are containers for storing data values. Use let, const, or var.

let name = "Geetika";
const PI = 3.14;
var age = 25;

Lesson 2: Functions

Functions let you reuse code:

function greet(name) {
  return "Hello, " + name;
}

Lesson 3: Conditionals

Conditional statements control the flow:

if (age > 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}
